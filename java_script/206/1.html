Оператор rest позволяет получить все лишние параметры в виде массива. Для этого в функцию добавляется еще один параметр (обязательно последним), перед которым пишутся 3 точки: function func(a, b, ...rest) { } Эти три точки и есть оператор rest, а имя переменной может иметь любое название, но обычно также принято называть его rest. Давайте вызовем нашу функцию с избыточным количеством параметров и посмотрим, что попадает в параметры a и b, а что - в остаток rest: func(1, 2, 3, 4, 5); function func(a, b, ...rest) { console.log(a); // выведет 1 console.log(b); // выведет 2 console.log(rest); // выведет [3, 4, 5] }
